## 🏗️ 技术架构

### 前端技术栈
- **React 18** - 现代化前端框架
- **TypeScript** - 类型安全的JavaScript
- **Vite** - 极速构建工具
- **SCSS + CSS Modules** - 模块化样式管理

### 功能模块
- **组件系统** - 20+ 模块化React组件
- **主题系统** - 暗黑/明亮主题切换
- **搜索系统** - 基于Fuse.js的智能搜索
- **评论系统** - Giscus + GitHub Issues双系统
- **SEO优化** - 结构化数据、sitemap、RSS

### 性能优化
- **代码分割** - 按需加载，8个独立chunk
- **懒加载** - 图片和组件懒加载
- **缓存策略** - 浏览器缓存优化
- **压缩优化** - Terser压缩，gzip支持
